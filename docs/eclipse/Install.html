• How to get Happah running in eclipse:
    • Open eclipse with fresh empty workspace
    • File>Import>Git>Project from Git
    • Choose Uri and input git url
    • Important: Change project directory from .../workspace/happah to .../happah so that the project is not in the eclipse workspace directory
    • Choose Import as general project and goto next and then finish
    • Now there should be the project happah in the project explorer
    • Right-click new>Convert to native C/C++ project
    • As project type choose Makefile Project>Linux gcc
    • Press finish and run qmake from terminal
  • Other helpfull settings:
    • To get rid of the eclipse error messages go to: Project>Properties>C/C++ Build>Discovery Options and choose gcc C++ Compiler instead of gcc C Compiler
